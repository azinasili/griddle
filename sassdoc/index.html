    grits-column

    @mixin grits-column($width: 100%, $gutter: map-get($grits, gutter), $shift: null) { ... }

    Description

    Create single column with gutter

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $width

    Width of column

    		Number100%
    $gutter

    Padding for column gutter

    		Numbermap-get($grits, gutter)
    $shift

    Shift source ordering of column

    		Numbernull

    Output

    CSS to create column with gutters

    Example

      .foo {
    @include grits-column(3/5, 1/3, 40px);
  }
      .foo {
    display: inline-block;
    font-size: 1rem;
    padding-left: 20px;
    padding-right: 20px;
    width: 60%;
    left: 33.33%;
    position: relative;
  }

    Requires

    • [function] _grits-is-gutter — Calculate gutter for row or column
    • [function] _grits-is-float — Check if number is a float
